# FLEET XRechnung Hybrid-PDF Viewer

Hier stellt EH Systemhaus die aktuelle Windows-Version des FLEET XRechnung Hybrid-PDF Viewers bereit.

!!! warning "Testversion"
    Diese EXE ist eine Testversion. Sie enthält keine Zugangsdaten und keine feste Datenbankkonfiguration.

## Aktuelle Testversion

Der automatische Build wird bei jedem Push auf den App-Branch `master` oder `main` erzeugt. Nach dem ersten erfolgreichen Build aktualisiert der App-Workflow diese Seite mit konkretem Dateinamen, Commit, Build-Zeitpunkt und SHA256-Prüfsumme.

## Download

Sobald der erste Build veröffentlicht wurde, liegt der Download im Release `latest` dieses Repositorys:

[Release `latest` öffnen](https://github.com/harzmann/fleet-erechnung-hybridviewer-docs/releases/tag/latest)

Die Einrichtung erfolgt nach dem Entpacken über `Einstellungen -> EHFleet-DB Einstellungen...`.
